# 捷报体育——体育羽毛球打卡

  "捷报体育——体育羽毛球打卡"是一款线上预约羽毛球馆和羽毛球教练的应用。
  
  【主要功能】
  1."羽毛球馆":罗列上海市优秀的羽毛球球馆,以及优秀的羽毛球教练。

  2."教练":通过羽毛球球馆对教练进行分类,羽毛球球馆与教练一对一展示。

  3."打卡":打卡展示关于羽毛球场馆在用户体验后，针对场馆和教练进行的图片或者文字打卡。
